# Password Generator & Strength Checker

A powerful web-based tool to generate secure random passwords with customizable options and real-time strength analysis.

## Features
- Generate random passwords instantly
- Adjustable password length (1-30 characters)
- Real-time password strength indicator
- Customizable character types:
  - Lowercase letters (a-z)
  - Uppercase letters (A-Z)
  - Numbers (0-9)
  - Special symbols (!^&*#@, etc.)
- One-click copy to clipboard
- Visual feedback on copy action
- Auto-generate on length adjustment
- Color-coded strength levels (Weak/Moderate/Strong)
- Clean and intuitive user interface

## Technologies Used
- HTML5
- CSS3
- Vanilla JavaScript
- Font Awesome Icons

## Installation
1. Clone the repository
```bash
   git clone https://github.com/ratneshchipre/Password-Generator-Strength-Checker.git
```
2. Navigate to the project directory
```bash
   cd Password-Generator-Strength-Checker
```
3. Open `index.html` in your browser
